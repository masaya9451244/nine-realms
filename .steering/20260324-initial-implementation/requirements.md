# 要求内容 - Nine Realms 初回実装

## 概要

数独 × ファンタジーRPGゲーム「Nine Realms」の初回実装。
Phaser 3 + TypeScript + Vite で構築するブラウザゲーム。

## ユーザーストーリー

- プレイヤーとして、オープニングでゲームの世界観を理解したい
- プレイヤーとして、ワールドマップで旅の進捗を把握したい
- プレイヤーとして、数独を解くことで敵を倒し先に進みたい
- プレイヤーとして、HPとダメージのスリルを感じながら数独を解きたい
- プレイヤーとして、ゴールドで装備・ヒントアイテムを買って強化したい
- プレイヤーとして、Realm9の魔王を倒してエンディングを迎えたい

## 実装する機能

1. **プロジェクトセットアップ** - Phaser 3 + TypeScript + Vite 環境構築
2. **数独コアロジック** - 問題生成・正解判定・候補数字計算
3. **タイトル画面** - ゲーム開始・セーブデータ確認
4. **オープニング演出** - 世界観説明テキスト
5. **ワールドマップ** - 9つのRealm表示・移動・ショップアクセス
6. **バトル画面** - 数独グリッド・HP・ダメージ・アイテムスロット
7. **ショップ画面** - 装備・ヒントアイテム購入
8. **Realmクリア演出** - クリア後の演出・テキスト
9. **エンディング** - 魔王撃破後のエンディング画面
10. **セーブ・ロード** - LocalStorageで進捗保存

## 受け入れ条件

- Realm1〜9まで通しでプレイできる
- 数独の正解判定が正しく動作する
- HP・ダメージシステムが機能する
- 装備・ヒントアイテムの効果が正しく反映される
- LocalStorageでセーブ・ロードができる
- デスクトップブラウザ（Chrome）で動作する
