# プロダクト要求定義書 - Nine Realms

## プロダクトビジョン

「Nine Realms」は、数独（Sudoku）とファンタジーRPGを融合したブラウザゲームです。
プレイヤーは勇者として9つの王国（Realm）を旅し、数独パズルを解きながら魔王を倒す旅に出ます。

## ターゲットユーザー

- 数独が好きで、ゲーム的な楽しさも求めている人
- ライトなRPGゲームが好きな人
- パズルゲームに達成感・物語性を求めている人

## 主要な機能

### 1. ワールドマップ
- 9つのRealmがマップ上に配置される
- 勇者がRealmを巡って進んでいく
- 現在地・クリア済みRealmが視覚的にわかる

### 2. 数独パズル（コア機能）
- 標準9×9の数独グリッド
- 難易度はRealmごとに上昇（Realm1=簡単 〜 Realm9=超難関）
- ボス戦では通常より難しいパズルが出現
- 正解で勝利、プレイヤーが入力をサポートするUI

### 3. バトルシステム
- Realmの道中に雑魚敵が出現
- ボスはRealm突破の鍵
- 雑魚戦：簡単な数独 → 勝利で先に進む
- ボス戦：難しい数独 → 勝利でRealm解放

### 4. ストーリー演出
- **オープニング**: ゲーム開始時に「この世界は9つのRealm（王国）に分かれており…」と世界観を説明するテキスト演出
- **最初のNPC（村人）**: Realm1の入り口で村人が「旅人よ、この先には9つのRealmが待ち受けている…」と補足説明
- Realmクリアごとに短いセリフ・演出
- 勇者とボスの掛け合いテキスト
- 魔王城（Realm9）クリアでエンディング

### 5. HPとダメージシステム

- 勇者はHPを持ち、バトル中にダメージを受ける
- ダメージ発生タイミング：
  - **時間経過**（例：10秒ごとに1ダメージ）
  - **ミス入力**（間違った数字を入力するたびにダメージ）
- HPはRealmレベルに応じて増加（難しいRealm = 長い戦闘時間に対応）
- HP0になるとバトル失敗、マップに戻って再挑戦可能

### 6. 装備システム

- 装備は3部位：**武器**・**防具**・**アクセサリー**
- 入手方法：ショップで購入 または ボス撃破時にドロップ
- 使い捨てではなく永続装備（付け替え可能）

| 部位 | 主な効果 |
|---|---|
| 武器 | 数独の空白マスが減る（問題が簡単になる） |
| 防具 | 最大HP増加 |
| アクセサリー | 時間ダメージ軽減、ミスダメージ軽減など |

### 7. ヒントアイテム・ショップ

- 雑魚敵・ボスを倒すとゴールド（G）を獲得
- ワールドマップのショップでヒントアイテムを購入（使い捨て）
- ヒントアイテムはバトル中に使用可能

| アイテム | 効果 | 価格 |
|---|---|---|
| 数字の光 | 選択マスの候補数字を表示 | 10G |
| 真実の目 | 間違いマスをすべて教える | 30G |
| 導きの手 | 1マスを自動入力 | 50G |

### 6. 進捗の保存
- LocalStorageでゲームの進捗を保存
- ページを閉じても再開できる

## 成功の定義

- 数独を解くことが「ゲームを進める」行為として自然に感じられる
- ストーリー演出により達成感が高まる
- Realm1〜9まで通しでプレイできる

## 制約事項

- ブラウザ動作（デスクトップ想定）
- バックエンドなし（LocalStorageのみ）
- 一人プレイのみ
