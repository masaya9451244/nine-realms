# ユビキタス言語定義 - Nine Realms

## ゲーム世界の用語

| 用語 | 定義 |
|---|---|
| Realm | この世界を構成する9つの王国。Realm1（草原）〜Realm9（魔王城）まで存在する |
| 勇者 | プレイヤーが操作するキャラクター |
| 魔王ナイン | 世界を9つのRealmに引き裂いた最終ボス。Realm9に君臨する |
| ボス | 各Realmの最後に待ち構える強敵。倒すとRealmが解放される |
| 雑魚敵 | Realm道中に出現する敵。倒すとゴールドを獲得できる |
| NPC | ゲーム内に登場する非プレイヤーキャラクター（村人など） |
| ゴールド（G） | ゲーム内通貨。敵を倒すと獲得し、ショップで使用する |

## 数独関連の用語

| 用語 | 定義 | コード上の名前 |
|---|---|---|
| グリッド | 9×9の数独盤面全体 | `grid: number[][]` |
| セル | グリッドの1マス | `cell` |
| 空白セル | 未入力のセル（値=0） | `emptyCell` |
| 初期セル | 最初から数字が入っているセル（変更不可） | `fixedCell` |
| 候補数字 | あるセルに入れられる可能性のある数字の一覧 | `candidates: number[]` |
| 難易度 | 空白セルの数で表す（多いほど難しい） | `difficulty: number` |
| 解答 | パズルの正解盤面 | `solution: number[][]` |
| ミス | 正解と異なる数字をセルに入力すること | `mistake` |

## バトル・システム用語

| 用語 | 定義 | コード上の名前 |
|---|---|---|
| HP | 勇者の体力。0になるとバトル失敗 | `hp: number` |
| 最大HP | HPの上限値。防具・アクセサリーで増加 | `maxHp: number` |
| 時間ダメージ | 10秒ごとに発生するダメージ | `timeDamage` |
| ミスダメージ | ミス入力時に発生するダメージ | `mistakeDamage` |
| バトル勝利 | 数独を正しく解くこと | `battleWin` |
| バトル失敗 | HPが0になること。マップに戻り再挑戦可能 | `battleLose` |

## 装備・アイテム用語

| 用語 | 定義 | コード上の名前 |
|---|---|---|
| 装備 | 勇者が身につけられるアイテム。3部位（武器・防具・アクセサリー） | `equipment` |
| 武器 | 空白セル数を減らす装備 | `weapon` |
| 防具 | 最大HPを増加させる装備 | `armor` |
| アクセサリー | 時間ダメージやミスダメージを軽減する装備 | `accessory` |
| インベントリ | 所持している装備の一覧 | `inventory: string[]` |
| ヒントアイテム | バトル中に使用できる消耗品。3種類存在する | `hintItem` |
| 数字の光 | 選択中のセルの候補数字を表示するアイテム | `numberLight` |
| 真実の目 | 間違いセルをすべてハイライトするアイテム | `truthEye` |
| 導きの手 | 空白セルを1つ自動入力するアイテム | `guidingHand` |

## UI用語

| 用語 | 定義 |
|---|---|
| ワールドマップ | 9つのRealmが配置されたゲームの全体マップ画面 |
| バトル画面 | 数独グリッドとHP・アイテムスロットが表示される戦闘画面 |
| ショップ | ワールドマップ上の共通購入画面。装備・ヒントアイテムを購入できる |
| ダイアログ | NPCとの会話やストーリーテキストを表示するUIボックス |
| HPバー | 現在HPを視覚的に示すバー |
| アイテムスロット | バトル画面下部に表示される所持ヒントアイテムの使用UI |
